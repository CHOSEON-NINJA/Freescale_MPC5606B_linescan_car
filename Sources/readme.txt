FreeMASTER Serial Communication Driver PowerPC MPC56xx
======================================================

This folder contains the MPC56xx platform-dependent part of FreeMASTER serial 
communication driver. Please see the User Manual for more information about 
how to configure and use the driver.


Driver example/test applications available in <install_path>/examples folder:
(<install_path> is typically C:\Program Files\FreeMASTER Serial Communication)

<install_path>/examples/MPC56xx/MPC5604BEVB
----------------------------------------------
- Freescale MPC56xxEVB development board, PPC5604B
- CodeWarrior Development Studio for MPC56xx, version 2.5
- The application was created using built-in CodeWarrior project creation
  wizard

<install_path>/examples/MPC56xx/MPC5604PEVB
----------------------------------------------
- Freescale MPC56xxEVB development board, PPC5604P
- CodeWarrior Development Studio for MPC56xx, version 2.5
- The application was created using built-in CodeWarrior project creation
  wizard
